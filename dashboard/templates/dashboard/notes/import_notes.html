Importer des Notes

Mettez à jour ou ajoutez des notes en téléchargeant un fichier Excel.

  1. Téléchargez le modèle Excel via le bouton "Exporter les notes" sur votre tableau de bord pour obtenir le format correct.
  2. Remplissez les colonnes MATRICULE, MATIERE, PERIODE, et NOTE.
  3. Les valeurs dans ces colonnes doivent correspondre exactement aux données de votre système pour être reconnues.
  4. La colonne ANNEE_SCOLAIRE sera ignorée ; les notes seront automatiquement associées à l'année scolaire active.
  5. Enregistrez votre fichier au format .xlsx ou .xls avant de le téléverser.
